# Project 3: Spots

### Description of Project and Functionality

Creating a multi-media website that you can interact with. The website will be interactive with the user when they hover over buttons and icons. The viewer will be able to look at this from desktop, tablet, and phone.

### Description of Technologies and Techniques used

While doing this project I would utilize the videos attached to the site to help me visualize creating. I would utilize DOT when I would get stuck to help me think and guide me in the right direction.
